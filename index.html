Sobre mí

Apasionado por el desarrollo de software, me encanta crear soluciones innovadoras que impacten positivamente en la vida de las personas. Mi enfoque se centra en escribir código limpio, eficiente y mantenible, siempre buscando aprender nuevas tecnologías y metodologías para mejorar mis habilidades. Disfruto enfrentando desafíos técnicos y trabajando en equipo para construir productos de calidad que marquen la diferencia.

Habilidades

Mi experiencia en desarrollo de software abarca múltiples tecnologías y lenguajes de programación, permitiéndome crear soluciones robustas y escalables. A continuación, detallo mis principales habilidades técnicas:

  • C++: Desarrollo de aplicaciones de alto rendimiento y sistemas embebidos.
  • C#: Desarrollo de aplicaciones de escritorio y servicios backend con .NET.
  • Java: Desarrollo de aplicaciones empresariales y Android.
  • HTML: Estructuración semántica de contenido web.
  • CSS: Estilización y diseño responsivo de interfaces web.
  • JavaScript: Desarrollo de aplicaciones web interactivas y dinámicas.
  • Angular: Desarrollo de aplicaciones web SPA (Single Page Applications).
  • SQL Server: Diseño y gestión de bases de datos relacionales.
  • React: Desarrollo de interfaces de usuario modernas y componentes reutilizables.

Experiencia

Sistema de Gestión de Inventarios

Desarrollo de un sistema de gestión de inventarios para una empresa de distribución de productos electrónicos. El objetivo principal fue crear una aplicación robusta y escalable que permitiera a los usuarios gestionar el inventario de manera eficiente, registrar transacciones y generar informes. Desarrollado con C#, .NET Framework y SOAP/REST.

Sistema de Pedidos de Comidas Rápidas

Desarrollo de un sistema de pedidos de comidas rápidas a domicilio, diseñado para optimizar y facilitar la gestión de pedidos, así como mejorar la eficiencia y experiencia tanto para los usuarios como para los restaurantes asociados. El sistema fue desarrollado con C++ y Mysql.

Sistema de Venta de Productos Web

Desarrollo de un sistema de venta de variedad de productos en la web. Para el desarrollo de este sistema se usó React, Node.js, express y Mysql.

Reportes y Visualización de Datos

Desarrollo de reportes y visualización de datos para el análisis de información de una comunidad estudiantil. Desarrollado con Power BI.

Sistema de Gestión de Bibliotecas

Desarrollo de un sistema de gestión de bibliotecas, cuyo propósito es facilitar la administración de libros, permitir a los usuarios realizar operaciones CRUD y mejorar la eficiencia de las bibliotecas. El sistema fue desarrollado utilizando Java.

Sistema de Registro de Estudiantes

Desarrollo de un sistema de registro de estudiantes, cuyo propósito es permitir a los estudiantes gestionar sus perfiles personales y administrar sus datos a través de un CRUD. El sistema fue desarrollado utilizando Angular, JSON, Node.JS y Express.
